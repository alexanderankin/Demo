Points can be appended to this graph by writing JSON to tcp socket 9838 with property graph. For example, some time series data, or other measurements.

echo '{"graph":{"x":0,"y":2}}' | nc localhost 9838
echo '{"graph":{"x":10,"y":3}}' | nc localhost 9838
echo '{"graph":{"x":20,"y":5}}' | nc localhost 9838
echo '{"graph":{"x":30,"y":5}}' | nc localhost 9838
